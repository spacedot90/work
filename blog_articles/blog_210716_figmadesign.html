웹페이지 URL 입력 한번으로 피그마 디자인 소스를 얻을 수 있다?

16 JULY, 2021

피그마 플러그인을 통해, 선 개발된(디자인 되어있지 않은) 페이지나, 평소 벤치마킹 하고싶었던 사이트를 디자인 파일로 받아볼 수 있다.평소에 참고하고 싶었던 UI를 해당 플러그인을 통해서 차용해 볼 수 있다.

Figma to HTML, CSS, React & more!

위 플러그인은 뷰포트 사이즈 기준으로 웹페이지를 굉장히 간단하게 디자인 산출물로 출력할 수 있게 도와준다.

** 이미지를 누르시면 플러그인 페이지로 이동합니다.

사용법

1.플러그인 설치-실행 후, 피그마로 출력하고 싶은 웹페이지 링크 복사(Cmd C)+붙여넣기(Cmd V)

2.뷰포트 사이즈 입력

3.Import. 끝!

** Chrome Extension도 제공하며, Json 파일을 업로드하여 디자인 파일 형식으로 받을 수도 있으니, URL과 Json 둘중 편한 방식을 사용하면 된다.

** 완벽한 컨버팅을 지원하지는 않는다. 중요한 것은 완성도 있는 사이트들이 어떤식으로 레이아웃 구성을 짰는지,간격값과 그리드는 어떻게 설정하는지 등의 구성 파악에 도움이 될 것 같다.
